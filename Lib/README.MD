# Gauge CSharp Lib

`Lib` is a dll that brings in Gauge's data types to C#. It is hosted in [Nuget](https://www.nuget.org/packages/Gauge.CSharp.Lib/).

You will need Gauge and Gauge-csharp plugin installed before installing `Lib`.
